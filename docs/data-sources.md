## 5. Chiến Lược Crawl Dữ Liệu

### 5.1 Vấn đề kỹ thuật
Hầu hết các trang web giáo dục VN (VnDoc, lop5.net, tailieuonthi.io.vn, khotientieuhoc.com) trả về **HTTP 403** khi fetch trực tiếp do:
- Kiểm tra User-Agent
- Rate limiting
- Bot detection (Cloudflare, etc.)

### 5.2 Giải pháp kỹ thuật khuyến nghị

```
Tier 1 — PDF trực tiếp (ưu tiên nhất):
  - Tìm link PDF trực tiếp trên Google (filetype:pdf + tên trường)
  - Download PDF → chuyển đổi thành text + trích xuất hình

Tier 2 — Browser Automation:
  - Dùng Playwright hoặc Puppeteer với headless Chrome
  - Mô phỏng hành vi người dùng thật (scroll, click)
  - Chờ JavaScript render xong trước khi extract

Tier 3 — Social Listening:
  - Crawl Facebook Groups (cần Facebook API hoặc manual)
  - YouTube transcript API cho video hướng dẫn
  - Webtretho/Lamchame không có anti-bot mạnh

Tier 4 — Manual:
  - Nhờ phụ huynh/giáo viên chia sẻ đề thi thật
  - Liên hệ trực tiếp các trung tâm luyện thi
```
